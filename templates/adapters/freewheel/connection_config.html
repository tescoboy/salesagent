Inventory Sync

Pull FreeWheel inventory taxonomy (sites, sections, series, video groups, ad unit packages, ad unit nodes) into the local cache. The product setup UI uses this cache so you can pick targeting from FreeWheel inventory without round-tripping to the FW API on every page render.

Reporting Sync

Pull a fresh delivery report from FreeWheel into the placement-stats cache — populates impressions, completed views, clicks, and spend for the AdCP delivery surfaces (get_packages_snapshot + get_media_buy_delivery). Defaults to today's pacing across all placements; usually finishes in 10-30 seconds.

Note: the Query Reporting API scope is still pending on most accounts — if you see "scope not granted", run the API Permissions check below to see the exact endpoints awaiting grant.

API Permissions

Probe FreeWheel's API to verify the configured credentials have every scope this adapter needs. Surfaces any IAM gaps before they bite mid-campaign — each check is a cheap read, nothing is mutated upstream.
