{# FreeWheel Adapter Product Configuration Template Fragment Included in the product add/edit form when the FreeWheel adapter is selected. Drives FreeWheelProductConfig — inventory selection, ad-unit package, content ratings, pricing. All inventory pickers load from the local freewheel_inventory cache via GET /api/tenant//adapters/freewheel/inventory. If a tenant hasn't run inventory sync yet, the pickers come back empty and the page links to the adapter settings. Expected context: - implementation_config: dict with current product implementation config - tenant_id: current tenant ID #}
